# University Management System

A full-stack university management application designed to manage students, faculty, courses, departments, and academic records efficiently through a centralized system.

The project helps automate university operations such as student registration, course management, attendance tracking, and result management.

---

## Features

### Student Features

* Student Registration & Login
* Course Enrollment
* View Attendance
* View Results & Academic Records
* Profile Management

### Faculty Features

* Faculty Login
* Manage Student Attendance

* Course Management

### Admin Features

* Add/Edit/Delete Students
* Add/Edit/Delete Faculty

## Architecture Overview

The application follows a layered architecture approach:

* Frontend handles user interactions and UI rendering
* Backend processes business logic and handles requests
* Database stores student, faculty, department, and course information

### Workflow

1. User logs into the system
2. Frontend sends requests to backend
3. Backend validates and processes data
4. Database stores and retrieves required information
5. Processed response is displayed to the user
